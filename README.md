# Code for epsSMASH paper
Repository for data analysis and figure generation in epsSMASH paper

## Repository structure

**Rmarkdown files in the main directory**\
Each .Rmd file contains the code for the figure mentioned in the file name. For figure 3, the GOMC, MFD, MiDAS_Global and HumGut .Rmd files must be run before Fig3.Rmd can be run. The Rmd files pull data from `data/` to create the figures.

**data/**\
Contains the data used by the Rmd files in the main directory for generating the figures of the paper. Metadata from other articles (Bundalovic-Torma et al., Dueholm et al., Microflora Danica... must be added to the respective directories from the articles) as well as results generated by epsSMASH on large genome collections (using https://github.com/zreitz/multismash to generate the all_regions.tsv+region_counts.tsv files). Some scripts refer to `gene_info.tsv` files, which contain gene-level information about epsSMASH BGC results. These were generated from the epsSMASH results using a custom multismash_parser python script (https://github.com/AOHD/multismash_parser) and have not been included in the Github repo due to their size. Some folders in data contain scripts for formatting or downloading data (such as the efetch script in `data/Bundalovic-Torma/`). `sphingomonadales/` contains several scripts which create symlinks for the 902 Sphingomonadales genomes from GTDB v220 and creates a tree from them. It also gene-calls them with Prodigal and the called genes are searched for the presenge of exosortase and PEP-CTERM genes with the HMM files in the folder. 

**figures/**\
Contains .svg files of the figures used in the paper

**bigscape/**\
contains a folder for each genome catalogue (Microflora Danica, MiDAS Global, Global Ocean Microbiome Catalogue and HumGut), which contains the scripts used to run BiG-SCAPE 2.0 on them. The BiG-SCAPE sequence similarity networks were analysed and annotated in Cytoscape v3.10.3 in order to create figure 4 and supplemental figures S3-S6 in the paper.

**supplemental_data/**\
Folder containing the .tsv file created in Fig3.Rmd which catalogs all exoPS BGCs detected by epsSMASH in the four genome catalogues.
